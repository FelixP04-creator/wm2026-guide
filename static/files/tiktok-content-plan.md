# WM 2026 — TikTok Content Plan & Video-Skripte

> Ziel: Reichweite aufbauen, Traffic auf die Website lenken, Newsletter-Abonnenten gewinnen.
> Alle Videos mit 0 € Budget erstellbar (CapCut Free, KI-Voiceover, Stock-Material).

---

## TikTok-Strategie

| Element | Strategie |
|---------|-----------|
| **Nische** | Fußball-WM + Reise + Fun Facts |
| **Frequenz** | 3–5 Videos/Woche (1× täglich ideal) |
| **Länge** | 15–60 Sekunden (kurz = bessere Completion-Rate) |
| **Ton** | Fakten-basiert, mit Humor, deutsch |
| **CTA** | "Link in Bio" → Website / Newsletter |
| **Hashtags** | #WM2026 #FussballWM #FIFAWorldCup #WM2026Deutschland |
| **Beste Posting-Zeit** | 18–20 Uhr (DE-Zeit), Wochenende 11–13 Uhr |

---

## Video-Formate (abwechselnd)

1. **Fun Fact** (15s) — "Wusstest du...?" mit schnellem Text-Overlay
2. **Listicle** (30s) — "Top 5 X" mit Bild-Slideshow
3. **News-Update** (20s) — Aktuelle Quali-Ergebnisse
4. **Teaser** (10s) — "Mehr Infos auf unserer Seite → Link in Bio"
5. **Behind-the-Scenes** (25s) — "So planst du deine WM-Reise"

### Workflow für 1 Video (15 Minuten)

1. **Skript kopieren** (hier aus diesem Doc) — 2 min
2. **Bilder/Videos sammeln** (Pexels, Pixabay, Google) — 5 min
3. **In CapCut zusammenbauen** — 5 min
4. **Text-Overlays + Musik** — 2 min
5. **Caption + Hashtags** — 1 min
